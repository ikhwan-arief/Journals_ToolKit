Crossref XML Utility

DOI XML Repair

Metadata follows the new XML file. Article DOIs remain exactly from the old XML, including prefixes and suffixes. Output timestamp is incremented for the Crossref update.

  • 1
    Old DataUpload old Crossref XML or fetch old DOI metadata from Crossref.
  • 2
    New XMLUpload the new Crossref XML that contains the metadata to keep.
  • 3
    MappingReview article matching so old article DOIs are retained correctly.
  • 4
    OutputCopy or download repaired XML for Crossref metadata update.
Quick Start Guide & How to Use
  1. Old Data: Upload your old Crossref XML file, or select Enter old DOIs to paste one DOI per line and click Fetch Full Crossref Metadata.
  2. New XML: Upload your new Crossref XML file containing the updated metadata.
  3. Configure URL Output: Choose whether to use article URLs from the new XML or override them manually.
  4. Verify DOI Mapping: Review the automatic article matching and correct pairs using the dropdown selections if needed.
  5. Generate XML: Click Regenerate after manual changes, then click Copy or Download.
