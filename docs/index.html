Toolkit pengelolaan jurnal

Alat praktis untuk memeriksa kesiapan jurnal.

Kumpulan aplikasi statis untuk membantu pengelola jurnal memeriksa OAI-PMH, memperbaiki XML DOI Crossref, menjalankan simulasi Arjuna, melakukan precheck DOAJ, dan menyusun roadmap evaluasi diri jurnal.

Buka aplikasi

DOI XML Repair

Perbaiki XML Crossref dengan mempertahankan DOI artikel lama, memetakan artikel, dan menghasilkan XML update siap salin atau unduh.

Buka aplikasi

Simulasi Penilaian Arjuna 2021

Isi indikator Arjuna 2021, hitung skor total, subtotal manajemen/substansi, dan cetak ringkasan.

Buka aplikasi
