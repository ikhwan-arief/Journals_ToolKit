Journals Toolkit

Kumpulan aplikasi statis untuk membantu pengelola jurnal memeriksa OAI-PMH, menyiapkan pra-akreditasi Arjuna, mensimulasikan skor Arjuna 2021, melakukan precheck DOAJ, dan menyusun roadmap evaluasi diri jurnal.

Validator OAI-PMH Jurnal OJS

Temukan endpoint OAI-PMH, validasi Identify/ListRecords, dan lihat rekomendasi teknis untuk harvester.

Buka aplikasi

Simulasi Penilaian Arjuna 2021

Isi indikator Arjuna 2021, hitung skor total, subtotal manajemen/substansi, dan cetak ringkasan.

Buka aplikasi

Pra Nilai Arjuna

Kumpulkan data awal akreditasi, URL edisi, dan URL PDF artikel sampel untuk ringkasan pemeriksaan.

Buka aplikasi

DOAJ Precheck

Checklist required/recommended, status kesiapan, reset, dan print.

Buka aplikasi

Evaluasi Diri Jurnal

Kuesioner mandiri untuk skor kapasitas jurnal, prioritas roadmap, rencana 12 bulan, salin hasil, TXT, dan cetak.
