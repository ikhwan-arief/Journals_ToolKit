Panduan singkat

  1. Masukkan URL halaman utama jurnal, contoh https://jurnal.example.ac.id/index.php/nama-jurnal.
  2. URL endpoint lengkap seperti /oai juga bisa digunakan.
  3. Tekan Enter atau klik tombol Periksa OAI-PMH.
  4. Lihat status, endpoint, metadata, publikasi, dan rekomendasi.

Pengaturan

Validator OAI-PMH Jurnal OJS

Masukkan URL website jurnal. Aplikasi akan mencoba menemukan endpoint OAI-PMH, memeriksa validitasnya, menampilkan metadata publikasi yang tersedia, dan memberi rekomendasi jika data tidak bisa ditarik oleh harvester.

Aplikasi ini berjalan langsung di GitHub Pages dan hanya membaca metadata publik dari endpoint OAI-PMH. Tidak membutuhkan login ke OJS. Versi GitHub Pages dapat memakai proxy CORS publik agar browser bisa membaca XML OAI-PMH dari server jurnal.
