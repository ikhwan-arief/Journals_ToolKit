Form Pemeriksaan Awal Akreditasi Jurnal

Pedoman 2021

Halaman ini membantu pengisian data minimum & lengkap untuk pemeriksaan awal akreditasi jurnal. Tidak ada akses baca ke URL yang Anda isikan—hanya pengecekan format URL.

Alur ringkas:
  1. Isi data pada bagian-bagian di bawah (minimal: URL jurnal, jenis pengajuan, dan daftar URL PDF artikel sampel).
  2. Klik Buat Ringkasan untuk melihat boks konfirmasi.
  3. Jika sudah benar, klik Konfirmasi & Salin → Buka alat bantu pemeriksaan awal. Tab baru akan terbuka dan Anda cukup tekan Ctrl+V (atau Paste) di sana.
A. Identitas & Akses
Harus menggunakan http atau https.
B. Kelembagaan Penerbit (opsional)
C. Penyuntingan & Manajemen (opsional)
D. Edisi & PDF Artikel Sampel (wajib)

Tambahkan edisi sesuai jenis pengajuan. Tiap edisi memuat 3 URL PDF artikel. URL harus dari domain resmi jurnal (bukan cache/mirror).

E. Penyebarluasan (opsional)
F. Plagiasi (opsional)
G. Kontak PIC (opsional)

Ringkasan untuk Konfirmasi

Jika tab baru tidak terbuka, izinkan pop-up untuk situs ini. Setelah tab terbuka, tekan Ctrl+V (atau Paste) di sana.
Catatan: Pemeriksaan ini mengikuti pedoman akreditasi yang berlaku. URL hanya diuji formatnya, tidak diakses.
